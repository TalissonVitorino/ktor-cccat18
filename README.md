# Ktor CCCAT18

Backend **Ktor** (Kotlin) com PostgreSQL desenvolvido durante o curso CCCAT18.

---

## Stack

| Biblioteca | Uso |
|---|---|
| Ktor Server | Framework HTTP |
| PostgreSQL | Banco de dados |
| kotlinx.serialization | JSON |
| Content Negotiation | Conversão automática de tipos |

---

## Como executar

```shell
./gradlew run
```

Disponível em `http://localhost:8080`
